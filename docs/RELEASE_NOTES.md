# Codex Usage Watch 0.1.0-beta.1

Codex Usage Watch is an experimental, private, non-blocking usage-awareness beta.
It estimates historical five-hour usage pressure from local Codex rate-limit
metadata. The value may be wrong and is not official quota, billing, or account
data.

The verified route is the checksummed standalone archive on Ubuntu 25.10 x86_64
with Codex CLI 0.144.4. Other Linux distributions may work but are unverified;
macOS is preview-only and Windows installation is unsupported. The native footer
and plugin marketplace are not part of this beta.

This beta adds shell-safe absolute hook commands, deterministic newest-transcript
selection, truthful missing-binary uninstall recovery, private vulnerability
reporting, and beginner-focused onboarding. It retains local SQLite state,
privacy-sanitized diagnostics, consent-first import, and fail-open hooks.

Before sharing feedback, review diagnostics and never attach transcripts,
prompts, responses, source code, local paths, `display.json`, or `state.sqlite3`.
Use the public bug/compatibility forms or the repository's private vulnerability
reporting form as appropriate.

Read `README.md`, `docs/INSTALL.md`, `docs/PRIVACY.md`, and `docs/SUPPORT.md`
inside the checksum-verified archive before installation.
