# Codex Usage Watch 0.1.0-beta.1

This beta provides a local, non-blocking five-hour-equivalent attention meter
for Codex. The supported route is a checksummed Linux x86_64 standalone binary
plus explicit user hooks.

It is not an official OpenAI quota, billing value, enforcement boundary, or
task-cost predictor. Prompt and response content is neither retained nor sent.
The native Codex footer adapter and plugin marketplace installation are not
part of this beta.

See `README.md`, `docs/INSTALL.md`, `docs/PRIVACY.md`, and `docs/SUPPORT.md`
inside the archive before installing.
